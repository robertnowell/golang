### 1.4: gif.go  
usage: 
```
go build gif.go  
./gif > filename.gif
```
Creates a simple black and white gif based on sinusoidal oscillation.  
  
### 1.5: fetch.go  
usage:  
```
go build fetch.go  
./fetch url  
```
Inspired by Unix curl, this program fetches data from a webpage and prints it to standard output.
